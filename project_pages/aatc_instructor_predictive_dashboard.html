Project Background & Description

The current dashboard is not designed in a user-friendly way for the planning team. The relationship between instructor qualifications and current resources is not shown. This is because the existing system should be executed based on certain rules (e.g. appropriate qualifications and no. of instructors). Also, the system also does not take into account the need for instructor rest.​

Due to a lack of inter-relationships between data and interactive functionalities, the system is deemed inefficient because it does not allow users to forecast the impact on instructor resources and related qualifications to deliver customer training should there be changes in course plans, simulator resources, customer specificities and course cancellations. ​

To address the gaps, the AATC instructor predictive dashboard acts as an accurate single source, and connects data in AATC to create inter-relationships between the data and interactive functionalities, providing insights on operational needs. Implementing this predictive dashboard will reduce operational steps and data silos.​
