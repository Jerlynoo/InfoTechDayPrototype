Project Background & Description

Many students find the transition from thier first to second year of Buisness Data Analytics(BDA) program challenging. Some might not yet realize how much Large Language Models(LLMs) can assit with data science projects.

Having experienced the same process myself, I can connect with the new group of studentd and create this guide to help make their journey smoother.

To make the transition smoother, this playbook will guide students on effectively using LLMs in their projects. It will introduce various prompt engineering techniques to help students find the most suitable approach for thier specific needs. By leveraging these tools, students can streamline thier projects and achieve better outcomes.
