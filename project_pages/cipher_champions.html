Project Background & Description

In our rapidly advancing technological society, cryptography is a fundamental component of cybersecurity, yet many IT students lack sufficient knowledge of this essential field. According to the OWASP Top 10, A02:2021 – Cryptographic Failures is the second most critical security risk category, with a maximum incidence rate of 46.44%. This highlights the need for understanding cryptography to make the right choices when implementing encryption, safeguarding sensitive data, and preventing breaches from weak cryptographic practices.

Cipher Champions is an immersive 2D dungeon crawler web game designed to bridge this knowledge gap by making learning cryptography engaging and accessible for students. Cipher Champions offers a unique solution to the often-tedious process of learning cryptography by incorporating interactive dungeon rooms filled with puzzles, thrilling password-cracking mini-games, and engaging character dialogues. Key features include intuitive drag-and-drop coding combat for beginners and challenging manual coding combat for advanced users. Through educational cutscenes and rewarding collectible certificates, students can master encryption, decryption, and password cracking in an enjoyable and motivating new way.
