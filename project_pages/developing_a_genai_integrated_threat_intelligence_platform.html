Project Background & Description

​ Every day, a security analyst is required to complete many different tasks related to operations, threat hunting, patching of vulnerabilities and many more. A key component to strengthen an organization’s security posture and streamline daily tasks like the ones mentioned earlier is threat intelligence. Threat intelligence can come from many different sources such as open-source security feeds, news pages and more, in many different formats. Manually collecting this data could prove to be overwhelming and time-consuming. Security specialists may not have the time to collect and analyse this data to filter out whatever is relevant to their organization as they have many different tasks to complete. A centralised platform to automatically ingest this data and perform analysis to filter data relevant to each organisation would reduce operational fatigue and allow for IT security professionals to spend their time completing other tasks.​

​ Our threat intelligence platform performs data ingestion from multiple feeds, which are analysed to determine relevancy to the organisation. Using this data, dashboards will be built to provide insights into the threat landscape on PowerBI. For more insights into specific threats, on-demand threat reports are generated using an OpenAI LLM powered chatbot and automatic reports generation.​
