Project Background & Description

Buddy is an enterprise chatbot developed for OCBC staff to provide accurate and confidential information while adhering to company policies. It addresses the challenge of ensuring employees can access reliable information quickly while fostering engagement through gamification. Users earn points by completing tasks, which they can redeem for attractive avatars, creating a fun and motivating experience.

Admins have a Backoffice application to manage rules, events, daily rewards, and avatars, ensuring the system remains dynamic and easy to update. Key features include seamless integration of gamification with the chatbot, avatar redemption, activity logs to track channel admin actions, and robust backend/frontend services.

Although still in the early stages of development, this project focuses more on admin control and users being able to redeem avatars for free.
