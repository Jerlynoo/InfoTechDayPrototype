Project Background & Description

Sharing important documents online can be risky, as there is always the possibility that files may fall into the wrong hands, be altered without permission, or remain accessible for longer than intended. These risks make it difficult to maintain control over who can view or use sensitive information, potentially leading to significant security issues.

File Link is a straightforward, web-based application designed to make file sharing more secure. It offers features such as watermarking, expiration dates, and download limits to help safeguard shared files. Watermarks prevent unauthorized copying, expiration dates ensure files don’t remain accessible indefinitely, and download limits allow you to control who can save the files. With File Link, sharing sensitive documents becomes more secure and easier to manage.

Additionally, this system is developed as a web-based platform that leverages AWS cloud services for running the website publicly processing the features. Content curators can also track readership performance after the system compiles the statistics when files are accessed, read, or edited by page.
